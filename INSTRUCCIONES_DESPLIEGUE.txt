# VERTEX AI HUB — Guía de despliegue
# Tiempo estimado: 20–30 minutos

═══════════════════════════════════════
PASO 1 — OBTENER TU API KEY DE ANTHROPIC
═══════════════════════════════════════

1. Ve a: https://console.anthropic.com
2. Crea una cuenta (con tu email)
3. Ve a "API Keys" → "Create Key"
4. Copia la key — empieza con "sk-ant-..."
5. Recarga créditos: mínimo $5 USD (te dura semanas con uso normal)

⚠️  NUNCA compartas esta key con nadie ni la pongas en el código directamente.


═══════════════════════════════════════
PASO 2 — SUBIR EL CÓDIGO A GITHUB
═══════════════════════════════════════

1. Ve a: https://github.com y crea una cuenta si no tienes
2. Crea un nuevo repositorio privado (botón "New") — llámalo "vertex-ai-hub"
3. Marca "Private" para que solo tú lo veas
4. Una vez creado, sube los 3 archivos:
   - index.html
   - api/chat.js
   - vercel.json

   Opción fácil para subir:
   a. Click en "Add file" → "Upload files"
   b. Sube index.html y vercel.json
   c. Para api/chat.js: crea la carpeta "api" primero (escribe "api/chat.js" en el nombre del archivo)


═══════════════════════════════════════
PASO 3 — DESPLEGAR EN VERCEL (hosting gratis)
═══════════════════════════════════════

1. Ve a: https://vercel.com
2. Crea cuenta con tu GitHub (botón "Continue with GitHub")
3. Click "Add New Project"
4. Selecciona tu repositorio "vertex-ai-hub"
5. Click "Deploy" — Vercel lo configura automáticamente


═══════════════════════════════════════
PASO 4 — CONFIGURAR TUS VARIABLES DE ENTORNO (LAS CONTRASEÑAS Y API KEY)
═══════════════════════════════════════

Esto es lo más importante — aquí pones tus datos reales de forma segura:

1. En Vercel, ve a tu proyecto → "Settings" → "Environment Variables"
2. Agrega estas 3 variables:

   Nombre: ANTHROPIC_API_KEY
   Valor:  sk-ant-XXXXXXXX (tu API key real de Anthropic)

   Nombre: PASSWORD_KEVIN
   Valor:  la contraseña que quieres para Kevin (ej: vertex2025kevin)

   Nombre: PASSWORD_WILLIAM
   Valor:  la contraseña que quieres para William (ej: vertex2025william)

3. Click "Save" en cada una
4. Ve a "Deployments" → Click en los 3 puntos del deployment → "Redeploy"
   (esto es necesario para que las variables tomen efecto)


═══════════════════════════════════════
PASO 5 — ACCEDER A TU APP
═══════════════════════════════════════

Vercel te da una URL como: https://vertex-ai-hub-XXXXX.vercel.app

¡Esa es tu app! Funciona en cualquier dispositivo, navegador y sin depender de Claude.ai.

OPCIONAL — Dominio propio:
Si quieres que sea https://app.vertexacademy.ec:
1. En Vercel: Settings → Domains → Add Domain
2. Escribe: app.vertexacademy.ec
3. Vercel te da un registro DNS — ve a donde compraste tu dominio y agrégalo
4. En ~5 minutos queda activo con HTTPS automático

═══════════════════════════════════════
¿PROBLEMAS? CAUSAS MÁS COMUNES
═══════════════════════════════════════

❌ "Contraseña incorrecta" → Revisa que pusiste bien la variable PASSWORD_KEVIN en Vercel
❌ "Error de API" → Revisa que la ANTHROPIC_API_KEY esté correcta y tenga créditos
❌ La app no carga → Revisa que los 3 archivos estén en GitHub (index.html, api/chat.js, vercel.json)
❌ Cambios no aparecen → Vercel redeploya automáticamente al hacer push a GitHub

Soporte: docs.anthropic.com | vercel.com/docs
